This example illustrates the use of API and the integration with prettyPhoto. prettyPhoto is built into the slider for ease of use but you can use other lightbox script as well if you want. Also, you can see that it's possible to navigate through the slides using external buttons. Another feature illustrated in this example is the slider's ability to display the caption only when the mouse is over the slide. Also, please note that the slideshow will pause when a modal box is opened.
